# ✅ MVP 清单

## 目标

> 第一版先完成一个会待机、会溜达、能被拖拽、看起来像活着的桌面宠物。
> 重点是**「陪伴感」闭环**，不追求完整养成系统。

---

## 第一版要做

### 1. 🖥️ 常驻桌面

- 启动后出现在桌面层
- 默认不出现在任务栏
- 支持基础窗口穿透

### 2. 🐾 基础行为

- 待机
- 随机溜达
- 到屏幕边缘折返
- 低频自发小动作：眨眼、打哈欠、发呆、看时间

### 3. 👆 交互反馈

- 可拖拽移动
- 松手后有简单反馈动作
- 支持点击反馈

### 4. 🌱 简单养成感

- 投喂一次基础食物
- 心情值有轻微变化
- 下次启动能保留基础状态

---

## 第一版明确不做

| 功能 | 原因 |
|---|---|
| 复杂数值系统 | 二期养成再加 |
| 打工 / 学习 / 生病 / 日记 | 超出陪伴感闭环范围 |
| 多宠物 | 状态管理复杂度跳跃 |
| 联网功能 | 不在本地桌面挂件核心路径上 |
| 换装系统 | 资源管线需先稳定 |
| 大量 UI 面板 | 保持无感存在感 |

---

## 完成标准

满足以下全部四点，即视为 MVP 成功：

- [ ] 挂在桌面上不碍事
- [ ] 偶尔自己动一下，不完全依赖用户点击
- [ ] 被拖拽和点击时有反应
- [ ] 用户会觉得「这小东西像是在活着」

---

## 后续扩展

| 阶段 | 功能 |
|---|---|
